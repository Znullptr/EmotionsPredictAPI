Welcome to My Emotions Predict API

This API makes emotion prediction from audio speech easier!

It is based on the RADVESS dataset, which contains a collection of emotional speech recordings.

Using advanced machine learning techniques, this API can analyze audio recordings and predict the corresponding emotions.

It is built with Django and follows the principles of a RESTful API.

The API is free to use and can be integrated into your applications to enhance emotion analysis capabilities.
